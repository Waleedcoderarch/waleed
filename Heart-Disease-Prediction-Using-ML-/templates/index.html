❤️ Heart Disease Prediction Project

I developed a Machine Learning–based system that predicts the presence and severity of heart disease with high accuracy.

To build this, I:

  • • Collected and combined a large medical dataset (1025 records).
  • • Applied data cleaning, feature selection, and PCA to improve model performance.
  • • Trained multiple models and optimized them with GridSearchCV and cross-validation.
  • • Used Gradient Boosting, which achieved 100% accuracy with predictions in just 0.003 seconds.
  • • Deployed the model with a Flask web application for easy real-time predictions.

This project showcases my skills in data science, machine learning, and web development, while addressing a real-world healthcare challenge.
