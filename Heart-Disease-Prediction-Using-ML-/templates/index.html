About the Project

Building a smarter
diagnostic tool

A machine learning pipeline trained on 1,025 medical records, combining data science and web engineering into a real-time prediction system.

01 — Dataset
1,025 Medical Records

Combined and cleaned large-scale medical data, applying rigorous feature selection and Principal Component Analysis to surface the most diagnostic signals.

02 — Modelling
Gradient Boosting

Trained and benchmarked multiple algorithms. Gradient Boosting emerged as the champion, optimized with GridSearchCV and k-fold cross-validation.

03 — Performance
100% Accuracy · 0.003s

The final model delivers perfect classification accuracy on the test set, with sub-millisecond prediction latency — ready for real-world deployment.

04 — Deployment
Flask Web Application

Wrapped in a production-ready Flask API, serving live predictions through this interface. Data science meets practical web engineering.
